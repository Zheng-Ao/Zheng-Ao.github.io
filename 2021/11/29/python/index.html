虚拟环境

pip

pip install 必须用国内镜像：

清华大学 https://pypi.tuna.tsinghua.edu.cn/simple/

阿里云 http://mirrors.aliyun.com/pypi/simple/

中国科技大学 https://pypi.mirrors.ustc.edu.cn/simple/

豆瓣(douban) http://pypi.douban.com/simple/

Eg: pip install -i https://pypi.tuna.tsinghua.edu.cn/simple scikit-learn==0.21.2

问题在于：每次新建一个项目，都要新建一个虚拟环境，重新安装相应的库。pip对虚拟环境的管理没有操作，它只能创建虚拟环境。

其实是可以的（使用先前配置的解释器），只不过确实不太方便（无法管理，必须一开始就想好）。

Python自带模块

pickle

pickle 模块提供了以下 4 个函数供我们使用：

  1. dumps()：将 Python 中的对象序列化成二进制对象，并返回；
  2. loads()：读取给定的二进制对象数据，并将其转换为 Python 对象；
  3. dump()：将 Python 中的对象序列化成二进制对象，并写入文件；
  4. load()：读取指定的序列化数据文件，并返回对象。
