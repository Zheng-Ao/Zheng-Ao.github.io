Algorithms

本书的目的：Technique

Although you can use this book as a “cookbook” for algorithms, you may someday encounter a problem for which you cannot readily find a published algorithm (many of the exercises and problems in this book, for example). This book will teach you techniques of algorithm design and analysis so that you can develop algorithms on your own, show that they give the correct answer, and understand their efficiency.

NP-complete：

If you are called upon to produce an efficient algorithm for an NP-complete problem, you are likely to spend a lot of time in a fruitless search. If you can show that the problem is NP-complete, you can instead spend your time developing an efficient algorithm that gives a good, but not the best possible, solution.

如何分析算法的正确性？——算法分析：第一弹

loop invariant（循环不变式）:

  1. Initialization: It is true prior to the first iteration of the loop.

  2. Maintenance: If it is true before an iteration of the loop, it remains true before the next iteration.

  3. Termination: When the loop terminates, the invariant gives us a useful property that helps show that the algorithm is correct.

其中1和2和数学归纳法（mathematical induction）一致。

Growth of Functions

2021/10/27

Asymptotic notation

本质：n趋于无穷大时比较函数的极限——“无穷大比阶”。在微积分里面算非常基础的内容。

方法很多，最简单的莫过于比值判别法（正项级数）、甚至洛必达……

大前提：函数渐近非负。

𝚯-notation

f(n) = 𝚯(g(n))

“同阶无穷大”——相差一个常数乘子。

We say that g(n) is an asymptotically tight bound for f(n).

O-notation

只有上界约束（大前提非负；洛后为0）。

显然，O-notation是𝚯-notation的父集。一个函数满足𝚯(g(n))则自然满足O(g(n))。

Distinguishing asymptotic upper bounds from asymptotically tight bounds is standard in the algorithms literature.

Ω-notation

只有下界约束（洛后无穷大）。

小写字母版

大写字母可以取等号，小写字母版是强约束（故𝚯无小写版）。
